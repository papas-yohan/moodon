## 🎨 디자인 스펙

### Grid Layout
```

이미지: 320 x 320px (6개)
라운드: 20px
배치: 3x2 그리드
```

### Highlight Layout
```
캔버스: 1080 x 1350px
메인: 700 x 700px
썸네일: 150 x 150px (4개)
라운드: 메인 30px, 썸네일 15px
```

### Simple Layout
```

이미지: 500 x 500px (1-2개)
라운드: 25px
배치: 세로 중앙
```
