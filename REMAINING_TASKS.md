## 🎯 남은 작업 (우선순위 순)

### 1순위: 솔라피 API 실제 연동 테스트 (30분 ~ 1시간)

#### 필요한 작업
```
1. 솔라피 계정 생성 및 본인인증
   - https://solapi.com 접속
   - 회원가입 및 본인인증
   - 발신번호 등록

2. API 키 발급
   - 콘솔 → API 설정
   - API Key 생성
   - API Key, API Secret 복사

3. 웹 UI에서 설정
   - http://localhost:5173/settings
   - 솔라피 API 탭
   - API 키 입력 및 저장

4. 테스트 크레딧 충전
   - 최소 5,000원 충전
   - 또는 신규 가입 무료 크레딧 사용

5. 실제 발송 테스트
   - 본인 전화번호로 테스트 발송
   - 메시지 수신 확인
```

#### 예상 비용
- 테스트 발송 (10건): 약 300원
- 초기 충전: 5,000원 권장
